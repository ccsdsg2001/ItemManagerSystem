# ItemManagerSystem
 本商品信息管理系统基于SpringBoot+Mybatis+Thymeleaf开发,提供了多种线上服务，
                                商家可以进入网站查看货源、查看物流信息，展示当前月收入，订单管理，注册登陆
                                等。后台超级管理员拥有最高权限，可以权限管理、用户管理，存储管理等。本项目
